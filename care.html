Уход за линзами

Инструкция по уходу за линзами

Мягкие дневные линзы

Ежедневно:

  1. Мойте руки с мылом, вытирайте безворсовым полотенцем
  2. Снимайте линзу, кладите на чистую ладонь
  3. Ополаскивайте свежим раствором
  4. Помещайте в контейнер с чистым раствором

Ночные жесткие линзы

Утренний уход:

  1. Снимать только с увлажняющими каплями
  2. Поместить в контейнер со специальным раствором для жестких линз
  3. Очистить пальцем, аккуратно протирая всю поверхность
  4. Промыть свежим раствором

Вечерний уход:

  1. Промыть раствором перед надеванием
  2. Осмотреть на повреждения
  3. При необходимости — дополнительная очистка

Контейнер для всех типов линз

Ежедневно:

  1. Выливать старый раствор
  2. Промывать свежим раствором
  3. Сушить на воздухе вверх дном
  4. Закрывать крышками

Ежемесячно:

  1. Менять на новый
  2. Не мыть водой из-под крана
  3. Не вытирать полотенцем

Растворы

  1. Использовать только свежий раствор
  2. Не смешивать старый и новый
  3. Соблюдать срок годности
  4. Для разных линз — разные растворы:
    • Для мягких — многоцелевые растворы
    • Для жестких — специальные растворы для ортокератологии

Перед использованием любых линз обязательно проконсультируйтесь с офтальмологом

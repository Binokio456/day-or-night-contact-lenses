Безопасное использование

Основные правила безопасности

Для всех типов линз:

  1. Чистота рук — всегда мойте руки с мылом перед любыми манипуляциями с линзами
  2. Регулярная замена — строго соблюдайте график замены линз и раствора (если линзы не однодневные)
  3. Контейнер — меняйте контейнер для линз каждый месяц
  4. Косметика — сначала надевайте линзы, потом наносите макияж

Особенности контактных линз:

  1. Время ношения — обычно 8-12 часов в день, не более
  2. Сушь глаз — при длительной работе за компьютером используйте увлажняющие капли
  3. Перерыв — минимум 1-2 дня в неделю носите очки для отдыха глаз
  4. Замена — строго по графику (ежедневные, двухнедельные, месячные)

Особенности ночных линз:

  1. Гигиена — ежедневная очистка специальным раствором для жестких линз
  2. Время ношения — обычно 6-8 часов сна
  3. Наблюдение — при покраснении, боли или ухудшении зрения немедленно снимайте линзы

Избегать таких ситуаций как:

  • Сон в дневных линзах (кроме специальных моделей)
  • Плавание в линзах без очков
  • Использование воду для промывания линз
  • Надевание поврежденные линзы
  • Использование просроченные растворы

Частые ситуации:

  1. Попадание соринки — снять линзу, промыть раствором
  2. Дискомфорт — снять линзы, дать глазам отдохнуть
  3. Воспаление — прекратить ношение, обратиться к врачу

Падение линзы на пол:

Ночные линзы:

  1. Промыть — специальным раствором для жестких линз
  2. Осмотреть особенно тщательно:
    • Повреждения края
    • Царапины на поверхности
    • Инородные частицы
  3. Очистить — дополнительно почистить пальцем с раствором

Если есть сомнения в целостности линзы и т.д. показать на следующем осмотре у врача

Контактные линзы:

  1. Промыть — раствором для линз (не водой!)
  2. Осмотреть — на свету проверить на:
    • Царапины
    • Надрывы
    • Загрязнения
  3. Продезинфицировать — положить в контейнер с раствором минимум на 4-6 часов

Если есть сомнения в целостности линзы и т.д. лучше линзы выкинуть

Важно: Любые линзы — медицинское изделие. Любые изменения режима использования согласовывайте с офтальмологом.

Перед использованием любых линз обязательно проконсультируйтесь с офтальмологом
